## 🎮 Game Description

- The game is played against the computer
- The computer randomly selects:
- Rock
- Paper
- Scissors
- The game is played for 5 valid rounds
- Invalid inputs repeat the round

## ⚙️ Functionality

- Random computer choice generation
- User input via prompt
- Case-insensitive input handling
- Round result calculation
- Score tracking
- Loop for 5 rounds
- Final winner announcement

## 🛠️ Technologies

- JavaScript
- HTML

## 📌 Notes

- The game runs entirely in the browser console
